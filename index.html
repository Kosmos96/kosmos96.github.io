Модули для Маркетплейса

В период работы в компании участвовал в разработке коммерческих модулей, опубликованных на Маркетплейсе 1С-Битрикс. Отвечал за техническую реализацию в рамках поставленных задач.

Корпоративный сайт с каталогом
Готовое решение для малого и среднего бизнеса: лаконичный адаптивный дизайн, встроенный каталог товаров с фильтрами, SEO-оптимизация и простая настройка. Позволяет быстро запустить сайт-визитку с возможностью лёгкой доработки.
Перейти
Перенос пользователей и заказов
Модуль для миграции пользователей и заказов между сайтами на 1С-Битрикс. Полностью заменяет соответствующие таблицы БД на целевoм сайтe. Можно выбрать, какие данные переносить: пользователей с их группами, заказы, корзины.
Перейти
Объединение заказов
Объединение товаров из нескольких заказов в один. Позволяет менеджерам за 3 клика собрать все позиции клиента (например, при дозаказах) в единый заказ, упрощая доставку и снижая риск ошибок. Исходные заказы можно отменить или удалить.
Перейти
Универсальный интернет-магазин
Готовое решение для быстрого запуска интернет-магазина на 1С-Битрикс: адаптивный дизайн, поддержка торговых предложений, избранное, сравнение товаров, гибкие скидки через модуль «Маркетинг» и несколько типов цен. Легко дорабатывается программистом, совместим с любыми модулями доставки и оплаты.
